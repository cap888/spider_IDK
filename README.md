# 🕷️ Spider Robot - Шестилапый робот на Arduino

Робот-паук с 6 лапами (12 сервоприводов SG90), управляемый через Bluetooth с помощью аналогового джойстика.

## Особенности

- ✅ **12 сервоприводов SG90** (6 лап × 2 сустава)
- ✅ **Bluetooth управление** через HC-05
- ✅ **Аналоговый джойстик** для интуитивного управления
- ✅ **Походка Tripod** (чередующиеся тройки лап)
- ✅ **Плавные движения** без рывков
- ✅ **Обратная кинематика** для точного позиционирования
- ✅ **Модульная архитектура** для лёгкой модификации

## Необходимые компоненты

### Аппаратное обеспечение

**Для пульта:**
- Arduino Nano ×1
- Модуль джойстика KY-023 ×1
- Bluetooth HC-05 ×1
- Аккумулятор 3.7V-5V

**Для робота:**
- Arduino Mega 2560 ×1
- PCA9685 (16-канальный PWM драйвер) ×1
- Сервопривод SG90 ×12
- Bluetooth HC-05 ×1
- Аккумулятор 7.4V-11.1V
- BEC 5V 3A (стабилизатор для сервоприводов)

### Библиотеки Arduino

Установите через Library Manager:
- **Adafruit PWM Servo Driver Library** (для PCA9685)

Встроенные библиотеки (не требуют установки):
- Wire.h (I2C)
- SoftwareSerial.h (для пульта)
- math.h

## Быстрый старт

### 1. Сборка пульта
1. Подключите джойстик к Arduino Nano (A0, A1, D2)
2. Подключите HC-05 к Nano (D3, D4)
3. Настройте HC-05 в режим Master (см. `docs/wiring_diagram.md`)

### 2. Сборка робота
1. Установите PCA9685 и подключите к Arduino Mega (SDA, SCL)
2. Подключите 12 сервоприводов к PCA9685
3. Подключите HC-05 к Mega (Serial1: D18, D19)
4. Настройте питание (5V 3A для сервоприводов)

### 3. Загрузка прошивок

**Пульт:**
```bash
# Откройте transmitter/transmitter.ino в Arduino IDE
# Выберите плату: Arduino Nano
# Загрузите
```

**Робот:**
```bash
# Откройте receiver/receiver.ino в Arduino IDE
# Выберите плату: Arduino Mega 2560
# Загрузите
```

### 4. Настройка Bluetooth
1. Включите робота
2. Включите пульт
3. Дождитесь подключения (LED на HC-05 начнёт мигать быстро)
4. Откройте Serial Monitor (115200 baud) для отладки

### 5. Управление
- **Джойстик вверх/вниз** - движение вперёд/назад
- **Джойстик влево/вправо** - стрейф влево/вправо
- **Кнопка джойстика** - зарезервировано

## Настройка

Все параметры в `receiver/Config.h`:

```cpp
// Геометрия лап
#define LEG_BASE_LENGTH 30    // Длина первого сегмента (мм)
#define LEG_JOINT_LENGTH 50   // Длина второго сегмента (мм)

// Параметры походки
#define STEP_HEIGHT 20        // Высота подъёма лапы (мм)
#define STEP_LENGTH 30        // Длина шага (мм)
#define STEP_FREQUENCY 2      // Частота шагов (Гц)

// Скорости
#define SERVO_MOVE_SPEED 60   // Скорость сервоприводов (°/с)
#define MAX_MOVE_SPEED 50     // Максимальная скорость движения
```

## Архитектура

### Принцип работы

```
[Джойстик] 
    ↓ (аналоговый сигнал)
[Arduino Nano + HC-05 Master]
    ↓ (Bluetooth пакет: X, Y, кнопки)
[Arduino Mega + HC-05 Slave]
    ↓ (обработка команд)
[Robot Controller]
    ↓ (расчёт походки)
[Gait Engine]
    ↓ (целевые позиции лап)
[Leg Kinematics] → обратная кинематика
    ↓ (углы сервоприводов)
[Servo Controller] → плавное движение
    ↓ (PWM сигналы)
[PCA9685]
    ↓ (12 каналов)
[12x SG90]
```

### Протокол Bluetooth

Формат пакета (6 байт):
```
[0xFF][Joy X 0-255][Joy Y 0-255][Buttons][Checksum][0xFE]
```

- **0xFF**: Стартовый маркер
- **Joy X**: Положение джойстика по X (128 = центр)
- **Joy Y**: Положение джойстика по Y (128 = центр)
- **Buttons**: Битовая маска кнопок
- **Checksum**: Контрольная сумма (сумма байтов & 0xFF)
- **0xFE**: Конечный маркер

### Походка Tripod

Лапы делятся на две группы:
- **Группа A**: Лапы 0, 2, 4
- **Группа B**: Лапы 1, 3, 5

**Фаза 1:**
- Группа A: в воздухе (swing) → перемещается вперёд
- Группа B: на земле (stance) → толкает тело назад

**Фаза 2:**
- Группа B: в воздухе (swing) → перемещается вперёд
- Группа A: на земле (stance) → толкает тело назад

Это обеспечивает стабильность и плавность движения.

## Устранение проблем

### Сервоприводы не двигаются
- ✅ Проверьте питание V+ на PCA9685 (должно быть 5V от внешнего источника)
- ✅ Проверьте I2C подключение (SDA → D20, SCL → D21)
- ✅ Убедитесь, что OE на PCA9685 подключен к GND

### Bluetooth не подключается
- ✅ HC-05 на пульте должен быть в Master режиме
- ✅ HC-05 на роботе должен быть в Slave режиме (по умолчанию)
- ✅ Пароли должны совпадать (по умолчанию: 1234)
- ✅ Адреса должны быть привязаны

### Робот двигается неправильно
- ✅ Откалибруйте сервоприводы (установите в 90°)
- ✅ Проверьте `INITIAL_POSITIONS` в Config.h
- ✅ Убедитесь в правильности подключения каналов PCA9685

Подробности: `docs/wiring_diagram.md`

## Безопасность

⚠️ **ВАЖНО:**
- НЕ подключайте V+ PCA9685 к 5V Arduino!
- Используйте отдельный источник питания 5V 3A для сервоприводов
- Суммарный ток 12 SG90 может достигать 3-4A
- Всегда объединяйте GND всех источников питания
