# Схема подключения шестилапого робота

## Компоненты

### Пульт управления (Передатчик)
- Arduino Nano
- Модуль джойстика KY-023
- Bluetooth HC-05
- Аккумулятор 3.7V-5V

### Робот (Приёмник)
- Arduino Mega 2560
- PCA9685 (16-канальный PWM драйвер)
- 12x Сервоприводы SG90
- Bluetooth HC-05
- Питание: 5V 3A (для сервоприводов) + 5V для Arduino

---

## 1. ПУЛЬТ УПРАВЛЕНИЯ

### Джойстик KY-023 → Arduino Nano
```
KY-023          Arduino Nano
------          ------------
VRx      →      A0            (ось X)
VRy      →      A1            (ось Y)
SW       →      D2            (кнопка, INPUT_PULLUP)
+5V      →      5V
GND      →      GND
```

### Bluetooth HC-05 → Arduino Nano
```
HC-05           Arduino Nano
-----           ------------
TX       →      D3            (SoftwareSerial RX)
RX       →      D4            (SoftwareSerial TX)
VCC      →      5V
GND      →      GND
```

**Важно:** HC-05 должен быть в режиме **Master** для пульта.

### Настройка HC-05 в режим Master
1. Подключите HC-05 к USB-TTL адаптеру
2. Войдите в AT-режим (удерживая кнопку при подаче питания)
3. Отправьте команды (скорость 38400):
   ```
   AT                  → OK
   AT+ROLE=1           → OK (Master mode)
   AT+PSWD="1234"      → OK (пароль)
   AT+BIND=<address>   → OK (адрес HC-05 на роботе)
   ```

## 2. РОБОТ (ПРИЁМНИК)

### Bluetooth HC-05 → Arduino Mega
```
HC-05           Arduino Mega
-----           ------------
TX       →      D19           (Serial1 RX)
RX       →      D18           (Serial1 TX)
VCC      →      5V
GND      →      GND
```

**Важно:** HC-05 на роботе должен быть в режиме **Slave** (по умолчанию).

### PCA9685 → Arduino Mega
```
PCA9685         Arduino Mega
-------         ------------
VCC      →      5V            (питание логики)
V+       →      5V EXTERNAL   (питание сервоприводов!)
GND      →      GND
SDA      →      D20           (I2C SDA)
SCL      →      D21           (I2C SCL)
OE       →      GND           (всегда включен)
```

**ВАЖНО:** 
- НЕ подключайте V+ к 5V Arduino!
- Используйте отдельный источник питания 5V 3A для V+
- Объедините GND всех источников питания

### Сервоприводы SG90 → PCA9685

Каждая лапа имеет 2 сервопривода:
- **Базовый сервопривод:** поворот лапы
- **Сервопривод сустава:** сгиб лапы

```
Лапа 0:
  Base  →  Канал 0
  Joint →  Канал 1

Лапа 1:
  Base  →  Канал 2
  Joint →  Канал 3

Лапа 2:
  Base  →  Канал 4
  Joint →  Канал 5

Лапа 3:
  Base  →  Канал 6
  Joint →  Канал 7

Лапа 4:
  Base  →  Канал 8
  Joint →  Канал 9

Лапа 5:
  Base  →  Канал 10
  Joint →  Канал 11
```

### Подключение SG90
```
SG90            PCA9685
----            -------
Оранжевый  →    Сигнал (канал 0-11)
Красный    →    V+ (5V внешнее)
Коричневый →    GND
```

## 3. СХЕМА ПИТАНИЯ

```
[Аккумулятор 7.4V-11.1V]
        |
        +---→ [BEC/регулятор 5V 3A] ---→ V+ PCA9685
        |                                     |
        |                                 12x SG90
        |
        +---→ [DC-DC 5V 1A] --------→ Arduino Mega
                                          |
                                      Bluetooth HC-05
```

**Рекомендуемые аккумуляторы:**
- LiPo 2S (7.4V) 1000-2000mAh
- Li-ion 18650 x2 (7.4V) 2000-3000mAh

**BEC (Battery Eliminator Circuit):**
- Turnigy BEC 5V 3A
- Или любой импульсный стабилизатор 5V 3A+

---

## 4. ПОРЯДОК СБОРКИ

### Шаг 1: Подготовка PCA9685
1. Установите перемычку на PCA9685 для I2C адреса 0x40 (по умолчанию)
2. Проверьте подключение к Arduino Mega (SDA, SCL, VCC, GND)

### Шаг 2: Подключение сервоприводов
1. Подключите все 12 сервоприводов к PCA9685 согласно схеме
2. Убедитесь в полярности питания (V+ от внешнего источника!)

### Шаг 3: Подключение Bluetooth
1. Подключите HC-05 к Serial1 (pins 18, 19)
2. Проверьте, что HC-05 в Slave режиме

### Шаг 4: Питание
1. Подключите BEC 5V 3A к V+ PCA9685 и GND
2. Подключите отдельный стабилизатор 5V к Arduino Mega
3. Объедините все GND

### Шаг 5: Тестирование
1. Загрузите прошивку receiver.ino на Arduino Mega
2. Откройте Serial Monitor (115200 baud)
3. Включите питание
4. Должно появиться: "Spider Robot starting..." → "All systems ready!"

---

## 5. НАСТРОЙКА BLUETOOTH ПАРА

### На роботе (HC-05 Slave)
Обычно не требует настройки. По умолчанию:
- Режим: Slave
- Скорость: 38400
- Пароль: 1234

### На пульте (HC-05 Master)
Обязательно настроить!
1. Подключите HC-05 к USB-TTL
2. Войдите в AT-режим
3. Узнайте адрес HC-05 на роботе:
   ```
   AT+ADDR?
   ```
4. Установите режим Master:
   ```
   AT+ROLE=1
   ```
5. Привяжитесь к адресу робота:
   ```
   AT+BIND=<адрес робота>
   ```
6. Установите пароль:
   ```
   AT+PSWD="1234"
   ```

---

## 6. КАЛИБРОВКА СЕРВОПРИВОДОВ

Перед первым запуском:

1. Загрузите тестовый скетч для калибровки
2. По одному подключайте сервоприводы
3. Установите каждый сервопривод в центральное положение (90°)
4. Закрепите лапы механически
5. При необходимости измените значения в `Config.h`:
   ```cpp
   const ServoCalibration INITIAL_POSITIONS[NUM_LEGS] = {
     {90, 90},  // Лапа 0 - измените при необходимости
     {90, 90},  // Лапа 1
     ...
   };
   ```

## 7. ОТЛАДКА

### Включение отладки
Раскомментируйте в `Config.h`:
```cpp
#define DEBUG_BLUETOOTH
#define DEBUG_SERVOS
#define DEBUG_KINEMATICS
#define DEBUG_GAIT
```

### Частые проблемы

**Сервоприводы не двигаются:**
- Проверьте питание V+ на PCA9685
- Проверьте I2C подключение (SDA, SCL)
- Убедитесь, что OE на PCA9685 подключен к GND

**Bluetooth не подключается:**
- Проверьте, что HC-05 на пульте в Master режиме
- Проверьте адреса и пароли
- Посмотрите LED на HC-05 (должен мигать быстро при подключении)

**Робот двигается неправильно:**
- Проверьте калибровку сервоприводов
- Проверьте правильность подключения каналов
- Отрегулируйте `INITIAL_POSITIONS` в Config.h

**Лапы двигаются в противоположных направлениях:**
- Инвертируйте углы в Config.h для проблемных лап

---

## 8. БЕЗОПАСНОСТЬ

⚠️ **ВАЖНО:**
- Никогда не подавайте питание на V+ PCA9685 от Arduino 5V pin!
- Всегда используйте отдельный источник питания для сервоприводов
- SG90 потребляют до 700mA каждый при нагрузке
- Суммарный ток 12 сервоприводов может достигать 3-4A
- Убедитесь в надёжном соединении всех контактов
- Используйте предохранитель на линии питания

---

## 9. СЛЕДУЮЩИЕ ШАГИ

После успешной сборки:
1. Откалибруйте все сервоприводы
2. Настройте Bluetooth соединение
3. Протестируйте движение каждой лапы
4. Запустите походку Tripod
5. Настройте параметры шага в Config.h
6. наслаждайтесь управлением! 🕷️
